AI Gym Tracker is an AI-powered workout monitoring system that uses MediaPipe Pose, OpenCV, and a Machine-Learning classifier to detect body posture, count exercise repetitions, and track workout performance in real time.

It also includes a Tkinter-based GUI to make it beginner-friendly and easy to use.
